Amenazas de Seguridad Informática

1. Ingeniería Social

La ingeniería social explota el factor humano manipulando a las personas para obtener accesos, información o realizar acciones que comprometan la seguridad.

Ejemplos comunes:

  • Phishing
  • Vishing
  • Smishing
  • Pretexting

Relevancia: Es difícil de detectar y suele ser la puerta de entrada para ataques más complejos.

2. Ransomware

El ransomware es un tipo de malware que cifra los datos de una organización o usuario y exige un pago para desbloquearlos.

Impactos principales:

  • Paralización de operaciones
  • Pérdida de información crítica
  • Costos económicos elevados

Tendencias actuales: Ataques dirigidos, doble extorsión y modelos de Ransomware-as-a-Service.

3. Vulnerabilidades Web

Las aplicaciones web están expuestas continuamente, lo que las convierte en un objetivo común para atacantes.

Vulnerabilidades comunes:

  • SQL Injection
  • Cross-Site Scripting (XSS)
  • Cross-Site Request Forgery (CSRF)
  • Configuraciones inseguras

Riesgos: Acceso no autorizado, robo de datos y compromiso del servidor.
